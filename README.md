# Rota Fluminense - Frontend

Aplicação frontend do projeto **Rota Fluminense**, desenvolvida como uma SPA (Single Page Application) utilizando **HTML, CSS e JavaScript puro**, consumindo uma API REST em Flask.

---

## Sobre o projeto

O objetivo da aplicação é listar locais turísticos do Rio de Janeiro e permitir interação com avaliações.

Este frontend consome o backend desenvolvido em Flask, exibindo os dados em um layout moderno utilizando **Bootstrap + CSS customizado**.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap 5

---

## 📸 Funcionalidades atuais

- ✅ Listagem de locais turísticos
- ✅ Renderização em formato de **cards**
- ✅ Consumo da API `/locais`
- ✅ Layout responsivo e moderno

## 🖥️ Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/rota_fluminense_frontend.git
```

### 2. Abra o arquivo `index.html` no navegador

- ✅ Não é necessário servidor local
- ✅ Não precisa instalar dependências

### Backend

Certifique-se de que o backend esteja rodando em http://127.0.0.1:5000/ antes de executar o frontend
