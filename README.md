# Rota Fluminense - Frontend

Aplicação SPA (Single Page Application) desenvolvida em HTML, CSS e JavaScript para consumo da API Rota Fluminense.

---

## Instalação

### 1. Clonar o repositório
```bash
git clone <url-do-repositorio>
cd rota_fluminense_frontend
```

### 2. Abrir o arquivo no navegador

Abrir o arquivo no navegador

### 3. Executar o Backend (Opcional)

Certifique-se de que o backend esteja rodando em http://127.0.0.1:5000/ caso queira persistir e consultar os dados.
